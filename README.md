# Space Shooter 🚀

A fast-paced, action-packed space shooter game built with **Godot 4.5.1**. Survive waves of enemies, dodge incoming fire, and rack up a high score!

## 🕹️ [Play the Game Now! (Click ME!)](https://nethphorn.github.io/space-shooter-game/)

---

## 🎮 How to Play

- **Move**: Arrow Keys or `WASD`
- **Shoot**: `Space`
- **Reset**: `R`
- **Quit**: `Esc`

## ✨ Features

- **Fast Gameplay**: Smooth movement and responsive controls.
- **Dynamic Enemies**: Various enemy types with unique movement patterns.
- **High-Quality Assets**: Beautifully crafted pixel art ships and environments.
- **Immersive Sound**: Background music and explosive sound effects (coming soon!).

## 🛠️ Technical Details

- **Engine**: Godot 4.5.1 (Forward Plus rendering)
- **Platform**: Web (HTML5) via GitHub Pages
- **CI/CD**: Automatic deployment using GitHub Actions

## 📦 How to Run Locally

1.  Clone this repository:
    ```bash
    git clone https://github.com/Nethphorn/space-shooter-game.git
    ```
2.  Open the `project.godot` file in **Godot 4.5.1+**.
3.  Press `F5` to start the game!
